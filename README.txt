This directory contains all the files which are needed to compile and run Progam on Windows or Linux systems.

Run and compile on Windows (from cmd) example:

compile.bat && run.bat C:\Users\Huawei\Downloads\Songs2.xml C:\Users\Huawei\Downloads\Songs1.csv

Run and compile on Linux (from terminal) example:

./compile.sh && ./run.sh "C:\Users\Huawei\Downloads\Songs1.csv C:\Users\Huawei\Downloads\Songs2.xml"
